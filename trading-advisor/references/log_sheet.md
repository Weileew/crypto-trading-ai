# Temporary Open/Closed Trade Sheet

Use this template to capture each trade decision as a lightweight entry. Each row should be a separate saved file under the skill workdir.

## File Name Convention
- `trade_YYYY-MM-DD_SYMBOL.md`
- Limit one trade idea per file for clarity.

## Template
```markdown
# Trade Log Entry

Date: YYYY-MM-DD
Asset: SYMBOL
Setup: swing / day
Direction: long / closed reason

Entry trigger:
- key price
- condition now or at entry

Risk / stop:
- hard stop level
- rationale

Target / exit:
- primary target and price
- partials at these levels if any

Rationale:
- one paragraph only

Actual result:
- exit price, outcome, notes
```

## Usage
- Append entries chronologically.
- At month end, roll into a compressed quarterly log.
- Include TokoCrypto liquidity notes when executed there.
